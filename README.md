# 課題用アプリ

[Next.js](https://nextjs.org) (App Router) ベースの課題用アプリです。

## 初期設定

このプロジェクトは [mise](https://mise.jdx.dev) でツール（Node.js / pnpm / prek）のバージョンを管理しています。

```bash
# Node.js と pnpm の設定
mise install
# Git Hooks の設定
prek install
```

## 開発

```bash
pnpm install
# 開発サーバー起動
pnpm dev # http://localhost:3000
```
